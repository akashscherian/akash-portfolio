Research project · 2023

Agri-Bot — smart agricultural robot

An IoT-based agricultural robot for smart irrigation and ML-driven plant-leaf disease detection. Published in IEEE Xplore.

Python IoT Machine learning Embedded systems

Overview

Agri-Bot is a small wheeled robot designed for use in field-scale agriculture. It performs two main jobs:

  • Smart irrigation — onboard soil-moisture sensing decides when and where to release water, avoiding the waste of blanket irrigation.
  • Leaf-disease detection — an onboard camera + a trained ML model classifies plant leaves and flags likely disease, letting farmers intervene before an outbreak spreads.

Publication

This work was peer-reviewed and published in IEEE Xplore as:

"Agri-Bot: A Smart Agricultural Robot for Smart Irrigation and Leaf Detection"
Published on IEEE Xplore.

What I contributed

  • End-to-end project delivery — concept, hardware-software integration, testing, and write-up.
  • Python software for sensor reading, decision logic, and ML-model inference on-device.
  • Bridging the hardware and software sides — a useful skill to have when later working on systems where the security boundary lives between firmware and application code.

Why I list this on a security résumé

Two reasons. First, it's a published, peer-reviewed piece of work — evidence that I can finish things and explain them in writing. Second, IoT and embedded systems are an increasingly large part of the security attack surface (operational technology, smart manufacturing, smart agriculture); having actually built one of these devices gives me real intuition for where they break.
